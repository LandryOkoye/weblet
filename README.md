# weblet

This is a web3 application where users can send and recieve tokens in the, their address undergo some validation to ensure secure transactions


## Setup
1. `npm install`
2. `npm run dev`

## How to Test Locally
1. **Create Wallet**: Click "Generate New Wallet".
2. **Fund Wallet**: 
   - Copy the address from the "Receive" section.
   - Go to [faucet.solana.com](https://faucet.solana.com) and airdrop SOL to your address (use Devnet).
3. **Send Token**:
   - You need a token Mint Address. If you don't have one, use a known Devnet mint or create one via CLI.
   - Enter recipient address and mint address.
   - Click Send.
